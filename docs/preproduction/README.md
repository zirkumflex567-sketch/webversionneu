# Preproduction Documentation Guide

This folder contains implementation contracts for the 20-hour Scherbenhimmel campaign scope.

## Start Here

1. `20H_DOCUMENTATION_INDEX.md`
2. `20H_AGENT_IMPLEMENTATION_PLAYBOOK.md`
3. `20H_AREA_PROGRESSION_MODEL.md`
4. Feature owner docs listed in the index

## Contract Rule

Preproduction contracts are authoritative for implementation behavior when they conflict with summary or legacy documents.

## Mandatory Rule Reminder

The campaign model is fixed:

`Hub -> Quest -> Area Instance -> Extraction -> Hub`

No seamless open-world interpretation is allowed for first-20h implementation.

## Localization Rule Reminder

All player-facing text must be implemented via stable localization keys with both German and English strings.
