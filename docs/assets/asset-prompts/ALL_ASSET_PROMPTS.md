# H-Town Combat / BIFA — All Asset Prompts

This is the repo-ready all-in-one asset promptbook for the 20-hour Scherbenhimmel scope.

## Prompt Formula

Every production prompt in this file uses this style prefix unless the asset is explicitly a grayscale wireframe:

```text
Premium wasteland cel-shaded dieselpunk dark-fantasy game art for H-Town Combat / BIFA / Scherbenhimmel; thick black outlines, graphic shape language, painterly cel shading, gritty industrial wear, weathered metal, oil, rust, moonglass glow, lantern-lit fog, high contrast, dramatic lighting, tactile materials, premium game-ready composition, grounded anatomy and believable hard-surface forms, cohesive art direction, no watermark.
```

Every production prompt also uses this negative prompt:

```text
Avoid photorealism, avoid modern clean sci-fi plastics, avoid generic fantasy armor, avoid low-detail mush, avoid text overlays unless explicitly requested, avoid watermarks.
```

For each asset below, combine:

```text
[STYLE PREFIX] + [ASSET-SPECIFIC PROMPT] + [NEGATIVE PROMPT]
```
